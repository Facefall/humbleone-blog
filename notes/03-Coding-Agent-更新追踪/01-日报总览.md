# Coding Agent 日报总览

本页是 coding agent 竞争追踪的事实层入口，负责回答两个问题：

1. 今天哪些官方 source 真有更新。
2. 今天哪些 source 没有更新，但仍值得保留注意。

它不承担深度竞争洞察任务。更深的结构性判断、研究点和行动点，统一放到周洞察页：

- [`02-周洞察总览.md`](./02-周洞察总览.md)

## 当前追踪范围

- OpenAI Codex GitHub releases / changelog
- CodeWhale 官方 changelog
- Reasonix 官方 changelog / releases
- Claude Code 官方 Anthropic 文章与工程内容

## 日报职责

每天的 `daily/YYYY-MM-DD.md` 应优先给出：

- 今日哪些 source 有变化
- 今日哪些 source 没变化
- 每个 source 的一句话更新判断
- 一句高信号但不展开的“今日关注点”

如果当天没有重大变化，也要直说，不强行编造成“有内容的一天”。
